# Drawing Trench Profiles for Digital Capture — Illustrator Guidelines

These profiles are digitized into structured data and 3D geological models. A few
consistent choices make that automatic and accurate; ambiguity forces guesswork
and errors. Please follow the points below when producing new drawings.

## Scale & measurement
- Use **one metric scale bar** (meters) per sheet. If an archival non-metric bar
  (e.g. PECK) must appear, place it separately and label it clearly
  **"archival reference — not for measurement."**
- Make the bar **long and finely ticked**, and place it **near the profiles**,
  not off in a corner.
- Best practice: also add a **depth scale down the side of each profile** with
  printed numbers (0, 0.5, 1.0 m …). Printed depths remove all guesswork.

## Origin & orientation
- Mark **x = 0 at the left edge** of each profile.
- Note the **compass direction** each face represents and which way distance
  increases (a small arrow is enough).

## Layer boundaries (most important)
- Draw each boundary as a **single, continuous, solid line of consistent
  weight.**
- Keep **boundary lines clearly bolder/darker than the fill hatching** inside
  layers, so each line can be followed on its own.
- Where two boundaries run close together, **keep a visible gap** — don't let
  them merge or touch.
- Draw each boundary's **actual shape** (dips, humps, pinch-outs). Avoid making
  layers look artificially parallel unless they truly are.

## Fill patterns & layer labels
- Choose **visually distinct hatch patterns** for different materials. Avoid
  several near-identical stippling densities that are hard to tell apart on a
  scan.
- **Label each layer directly on the profile** with its material name or a short
  code — not only via a legend swatch.
- Include a **legend** where every pattern maps to exactly one material, with no
  duplicates or near-duplicates.

## Features (stones, carbon lenses, pits, floor)
- Place each **feature label on the feature**, or connect it with a **leader
  line**. Don't let a label float away from what it names.
- Use a **consistent convention per feature type** (e.g. carbon lenses always the
  same bold line; stones always outlined shapes).
- Add a short **feature key** in a corner (e.g. "F1: carbon lens; F2: stone
  cluster") with matching labels on the drawing.

## Metadata (fixed positions)
- Put **trench label, face names, creator, date, and drawing number** in the
  **same corner every time**, and **print** them (no cursive).

## Scanning / digital capture
- Scan at **300 DPI or higher**, **high contrast**, **minimal skew**, with the
  **full scale bar in frame.**
- Save **lossless** (PNG or TIFF), not low-quality JPEG.
- If drawing digitally, **export vector (SVG or PDF).** Vector boundaries carry
  real coordinates and are by far the easiest to digitize accurately.

## Quick pre-submission checklist
- [ ] One metric scale bar, near the profiles, finely ticked
- [ ] Depth numbers along each profile's side (if possible)
- [ ] x = 0 marked and compass direction noted per face
- [ ] Boundaries bold, continuous, distinct from fills
- [ ] Distinct fill patterns; each layer labeled on the profile
- [ ] Legend maps one pattern → one material
- [ ] Feature labels on/pointing to their features; feature key present
- [ ] Metadata printed in the usual corner
- [ ] High-res, high-contrast, lossless scan (or vector export)
